Simple_Camera_App is an iOS app written using Swift 5.1 in the Xcode 11.1 IDE demonstrating how to use the camera on an iPhone to take and display pictures, as well as how to access, and display photos in, the photo library of an iPhone.

Run Instructions:
	1. Download Simple_Camera_App folder
	2. Open Simple_Camera_app.xcodeproj in Xcode IDE
	3. Click the Run icon to build and run the app in the iOS simulator.
	4. To run on an iPhone, connect the iPhone to the laptop via USB cable and ensure that the device indicated in the upper left icon 'Simple_Camera_App > [Device]' is your device. Click the run icon. 
	5. While running, the 'Take_Photo' button in the app screen brings up the camera and displays the captured photo and the 'Select_Photo' button does the same with a photo from the photo library. 
	6. To quit the simulator or stop the app running on your phone, click the stop button or hit Cmd+Q.
